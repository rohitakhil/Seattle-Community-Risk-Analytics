# RapidMiner Folder

This folder contains machine learning workflows, process screenshots, and model outputs developed using RapidMiner.

## Models Included

### Linear Regression
Predict neighborhood poverty levels.

### Decision Tree
Identify key decision factors contributing to economic vulnerability.

### Random Forest
Improve prediction stability through ensemble learning.

### K-Nearest Neighbors (KNN)
Classify neighborhoods into poverty categories.

### K-Means Clustering
Segment neighborhoods into similar socioeconomic groups.

## Purpose

These models were used to validate findings generated through Python and Tableau analyses.
