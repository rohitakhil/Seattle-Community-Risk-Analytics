# Data Folder

This folder contains the source datasets used throughout the Seattle Community Risk Analytics Platform project.

## Contents

### Seattle_Neighborhood_Data.xlsx

Neighborhood-level socioeconomic data for Seattle, including:

* Poverty rates
* Employment rates
* Labor-force participation
* Not-In-Labor-Force (NILF) metrics
* Household structure indicators
* Demographic characteristics

## Purpose

The dataset serves as the foundation for:

* Exploratory Data Analysis
* Statistical Testing
* Regression Modeling
* Classification
* Clustering
* Dashboard Development
